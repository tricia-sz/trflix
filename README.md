## 🖥️ Como rodar este projeto 🖥️


### Execução:

1. Clone este repositório:

   ```sh
   git clone git@github.com:tricia-sz/trflix.git
   ```

2. Acesse o diretório do projeto:

   ```sh
   cd trflix
   ```



3. Instalando as libs.
  ```sh
  npm install
   ```
3. Rodando o projeto no localhost.
  ```sh
  npm dev
   ```
